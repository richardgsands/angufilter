### Getting Started
Download the code, and include the mydirective.js file in your page. Then add the mydirective module to your Angular App file, e.g.
```html
var app = angular.module('app', ["mydirective"]);
```

### Local Usage

```html
<mydirective  prop1="val1"
              prop2="val2"/>
```
